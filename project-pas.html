Mobile Phone Pricing Classification

Mobile Phone Pricing Classification is a supervised machine learning project aimed at predicting the price category of mobile phones based on various hardware specifications. Developed during my internship at Unified Mentor, this project utilizes classification algorithms to categorize phones into low, medium, high, and very high price ranges.

The project follows a comprehensive ML pipeline, from data cleaning and exploratory data analysis (EDA) to feature engineering and model evaluation, ensuring high accuracy and robust performance.

Key Features

Exploratory Data Analysis (EDA)

In-depth analysis of hardware features like RAM, Battery Power, and Clock Speed to identify correlations with price.

Feature Engineering

Processing and scaling numerical data to improve model convergence and performance using Scikit-Learn.

Model Training

Comparison of multiple algorithms including Logistic Regression, Decision Trees, and Random Forest for optimal results.

Model Evaluation

Performance tracking using accuracy scores, confusion matrices, and precision-recall metrics.

The project successfully categorized mobile phone prices with high accuracy, demonstrating the power of machine learning in market research and product positioning.
